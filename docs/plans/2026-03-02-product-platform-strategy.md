## ステータス: 戦略確定（有料→無料スイッチ / 価格.com型）

## 1. プロダクトの本質（再定義）

楽々省エネ計算は「BEI計算ツール」ではない。

> **省エネ適合のための「建材選定 × 公式計算 × メーカー紹介」プラットフォーム**

計算は手段。価値の核心は:
- 建築士が**実際の製品を選んで**省エネ適合を達成できること
- 選んだ製品を**メーカーに直接つなぐ**こと
- この全プロセスが**公式APIによる正式な計算結果**で裏付けられること

## 2. 戦略的アセット

### パートナーシップ

| パートナー | 関係 | 戦略的価値 |
|-----------|------|-----------|
| **YKK AP** | 会長が知り合い | 窓サッシ国内シェア1位。製品DBの優先アクセス、スポンサー契約、販売店ネットワークへのツール展開 |
| **パナソニック テクノストラクチャー** | 知り合い | 空調・照明・太陽光・蓄電池。テクノストラクチャー加盟工務店（全国約350社）への展開チャネル |

### 重要な事業環境変化: YKKグループ統合

- **2025年11月**: パナソニックHDがパナソニック ハウジングソリューションズ(PHS)の株式80%をYKKに売却する契約を締結
- **2026年3月末**: 譲渡完了予定
- **2026年4月**: YKK傘下で新体制スタート（パナソニックブランドは維持）
- **合算売上**: YKK AP ¥5,616億 + PHS ¥2,536億 = **約8,000億円超**
- **テクノストラクチャーもYKKグループに**

**→ Senaの2つのコネクション（YKK AP会長 + テクノストラクチャー）が同一グループになる。**
**→ 別々の提案ではなく「YKKグループへの統合提案」として楽々省エネ計算を位置づける。**
**→ グループシナジーを示すデジタル施策として、窓(YKK AP) + 設備(パナソニック)を一気通貫で推薦するプラットフォームは極めて高い戦略的価値を持つ。**

### 技術的アセット

| アセット | 内容 |
|---------|------|
| lowenergy.jp API | 国交省の公式計算エンジン。無料・無認証。結果は審査提出に使える正式なもの |
| WEBPRO計算ロジック | MIT License でGitHub公開。自前計算エンジンとしても利用可 |
| 製品性能データ | YKK AP: Excel公開、LIXIL: 性能値参照システム公開、三協アルミ: PDF公開 |

### 市場環境

- 2025年4月の省エネ義務化で**44.5万棟が新規対象**
- 外注はパンク状態（需要 > 供給）
- AI活用のBEI計算ツールは**市場に存在しない**（ブルーオーシャン）
- 競合はすべて「計算代行」止まり。製品推薦まで一気通貫で行うサービスはゼロ

| 原則 | 適用箇所 |
|------|---------|
| **#1 選択肢 > 自由入力** | 全ステップで製品・スペックをボタン/カードで選択。手入力は最終手段 |
| **#2 AI出力にアクションボタン** | 改善提案に [適用して再計算] [他の選択肢] [スキップ] |
| **#5 チャットUIを安易に採用しない** | AIは構造化された提案カード。チャットではない |
| **#6 ツールは脇役** | 計算→製品選定→提出を最短で。余計な機能なし |
| **#7 派手より楽** | 製品を選ぶだけでBEI適合。認知負荷最小 |
| **#10 デフォルト最適化** | 地域×用途で最適な製品をデフォルト選択済み |

## 4. 紹介（リファラル）システム

### フロー

```
建築士が製品を選定
    ↓
[この製品でメーカーに見積依頼] ボタン
    ↓
メーカー側（YKK AP / パナソニック等）に通知
  - 建物情報（用途・面積・地域）
  - 選定製品と数量概算
  - 建築士の連絡先
    ↓
メーカー営業 or 代理店が建築士に連絡
    ↓
成約時: 紹介手数料（2-5%）
```

### 紹介手数料の試算

| 製品カテゴリ | 1案件あたり金額 | 紹介料率 | 紹介料/案件 |
|-------------|----------------|---------|------------|
| 窓サッシ | 50万〜200万円 | 2-3% | 1万〜6万円 |
| 空調設備 | 100万〜500万円 | 1-2% | 1万〜10万円 |
| 照明設備 | 30万〜150万円 | 2-3% | 0.6万〜4.5万円 |
| 太陽光発電 | 100万〜300万円 | 2-3% | 2万〜9万円 |
| **合計/案件** | — | — | **4.6万〜29.5万円** |

月間10案件の紹介で **46万〜295万円/月** の紹介収益。

### YKK AP・パナソニックとの交渉ポイント

**2026年4月以降はYKKグループ統合提案:**

> YKK APとパナソニック住宅事業が同一グループになるため、別々の提案ではなく
> 「グループのデジタルマーケティングプラットフォーム」として1つの提案で展開。

**YKK AP:**
- 楽々省エネ計算の「おすすめ製品」としてAPWシリーズを優先表示
- 楽々側が求めるもの: 初期開発費の一部負担（共同開発パートナーとして）+ 製品データ優先提供 + 販売ネットワーク（代理店26社、リノベの会351社）でのツール告知
- YKK APにとってのメリット: 省エネ計算の段階で自社製品が選ばれる = 最上流での顧客獲得
- 製品選択データ・リード情報をダッシュボードで提供

**パナソニック テクノストラクチャー:**
- テクノストラクチャー加盟工務店（約350社）に楽々省エネ計算を推奨ツールとして展開
- 空調・照明・太陽光・蓄電池でパナソニック製品を優先表示
- テクノストラクチャーの構造計算 + 楽々省エネの省エネ計算 = ワンストップ
- 加盟店はスポンサー契約後に完全無料化 → 導入障壁ゼロ（外注費年間30万円/社の削減効果）

## 5. 価格戦略: 有料→無料スイッチ（価格.com型への段階的移行）

### 戦略的判断（2026-03-03 確定）

**長期ゴール:** 価格.com型の両面市場プラットフォーム（建築士無料 + メーカー課金）。
**ただし初日から無料にはしない。** 有料で始めてキャッシュと実績を作り、スポンサー獲得後に無料化する。

→ 詳細な比較分析: `docs/plans/2026-03-03-free-vs-paid-analysis.md`

### Phase A: 有料ローンチ（〜100社到達まで）

| 機能 | 料金 |
|------|------|
| BEI計算（公式API）+ AI製品推薦 + PDF出力 | **月額 ¥9,800** |
| 改善シミュレーション | 含む |
| メーカー見積依頼 | 含む |
| サークル会員（¥5,000/月） | **含む（追加料金なし）** |

**なぜ ¥9,800 で売れるか:**
- 外注 8〜15万/件 → 月1件で元が取れる（1/10〜1/15）
- SAVE-建築 初年46.2万 → 年11.8万は1/4
- 「使わない理由がない」は有料でもほぼ変わらない

**目的:**
1. キャッシュフロー確保（月50社で49万円）
2. 課金ユーザー数 = スポンサー交渉の武器
3. 「お金を払ってでも使いたい」の証明

### Phase B: スポンサー獲得（100社到達時）

100社の課金実績を持ってYKK AP・パナソニックに提案:

> 「100社の設計事務所が月額課金して使っています。
> 御社の製品を優先表示しませんか？」

無料ユーザー100社より、課金ユーザー100社の方が**稟議を通す説得力が圧倒的に上**。

### Phase C: 無料スイッチ（スポンサー契約後）

スポンサー契約が決まったら建築士側を**完全無料**に切り替え。

- 既存有料ユーザーに「無料になりました」と通知 → 離脱ゼロ + 口コミ爆発
- テクノストラクチャー380社展開をここで開始 → 導入障壁ゼロ
- ネットワーク効果が発動 → 両面市場として加速

### メーカー向け収益（Phase C以降）

| **パートナー掲載料** | 30〜100万円/月/社 | AI推薦での優先表示、製品データの充実表示 |

## 7. 競合に対する参入障壁（Moat）

| 障壁 | 内容 |
|------|------|
| **メーカーパートナーシップ** | YKK AP会長、パナソニックテクノストラクチャーとの直接関係。他社が容易に真似できない |
| **製品DB** | パートナーから優先提供される性能データ。公開情報+非公開データの組み合わせ |
| **AI推薦の学習データ** | 計算実績が溜まるほど「この条件ならこの製品」の精度が上がる |
| **ネットワーク効果** | 建築士が増える→メーカーが参加したがる→製品が充実→建築士が増える |
| **省エネ義務化の波** | 2025年義務化で需要爆発中。先行者優位が極めて大きい |

## 8. リスクと対策

| リスク | 影響 | 対策 |
|--------|------|------|
