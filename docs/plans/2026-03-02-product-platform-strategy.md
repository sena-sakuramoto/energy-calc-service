## 1. プロダクトの本質（再定義）

楽々省エネ計算は「BEI計算ツール」ではない。

> **省エネ適合のための「建材選定 × 公式計算 × メーカー紹介」プラットフォーム**

計算は手段。価値の核心は:
- 建築士が**実際の製品を選んで**省エネ適合を達成できること
- 選んだ製品を**メーカーに直接つなぐ**こと
- この全プロセスが**公式APIによる正式な計算結果**で裏付けられること

## 2. 戦略的アセット

### パートナーシップ

| パートナー | 関係 | 戦略的価値 |
|-----------|------|-----------|
| **YKK AP** | 会長が知り合い | 窓サッシ国内シェア1位。製品DBの優先アクセス、紹介手数料交渉、販売店ネットワークへのツール展開 |
| **パナソニック テクノストラクチャー** | 知り合い | 空調・照明・太陽光・蓄電池。テクノストラクチャー加盟工務店（全国約380社）への展開チャネル |

### 技術的アセット

| アセット | 内容 |
|---------|------|
| lowenergy.jp API | 国交省の公式計算エンジン。無料・無認証。結果は審査提出に使える正式なもの |
| WEBPRO計算ロジック | MIT License でGitHub公開。自前計算エンジンとしても利用可 |
| 製品性能データ | YKK AP: Excel公開、LIXIL: 性能値参照システム公開、三協アルミ: PDF公開 |

### 市場環境

- 2025年4月の省エネ義務化で**44.5万棟が新規対象**
- 外注はパンク状態（需要 > 供給）
- AI活用のBEI計算ツールは**市場に存在しない**（ブルーオーシャン）
- 競合はすべて「計算代行」止まり。製品推薦まで一気通貫で行うサービスはゼロ

### UI設計原則の適用

| 原則 | 適用箇所 |
|------|---------|
| **#1 選択肢 > 自由入力** | 全ステップで製品・スペックをボタン/カードで選択。手入力は最終手段 |
| **#2 AI出力にアクションボタン** | 改善提案に [適用して再計算] [他の選択肢] [スキップ] |
| **#5 チャットUIを安易に採用しない** | AIは構造化された提案カード。チャットではない |
| **#6 ツールは脇役** | 計算→製品選定→提出を最短で。余計な機能なし |
| **#7 派手より楽** | 製品を選ぶだけでBEI適合。認知負荷最小 |
| **#10 デフォルト最適化** | 地域×用途で最適な製品をデフォルト選択済み |

## 4. 紹介（リファラル）システム

### フロー

```
建築士が製品を選定
    ↓
[この製品でメーカーに見積依頼] ボタン
    ↓
メーカー側（YKK AP / パナソニック等）に通知
  - 建物情報（用途・面積・地域）
  - 選定製品と数量概算
  - 建築士の連絡先
    ↓
メーカー営業 or 代理店が建築士に連絡
    ↓
成約時: 紹介手数料（2-5%）
```

### 紹介手数料の試算

| 製品カテゴリ | 1案件あたり金額 | 紹介料率 | 紹介料/案件 |
|-------------|----------------|---------|------------|
| 窓サッシ | 50万〜200万円 | 2-3% | 1万〜6万円 |
| 空調設備 | 100万〜500万円 | 1-2% | 1万〜10万円 |
| 照明設備 | 30万〜150万円 | 2-3% | 0.6万〜4.5万円 |
| 太陽光発電 | 100万〜300万円 | 2-3% | 2万〜9万円 |
| **合計/案件** | — | — | **4.6万〜29.5万円** |

月間10案件の紹介で **46万〜295万円/月** の紹介収益。

### YKK AP・パナソニックとの交渉ポイント

**YKK AP:**
- 楽々省エネ計算の「おすすめ製品」としてAPWシリーズを優先表示
- 見返り: 製品DBの優先提供 + 紹介手数料 + 販売店ネットワークへの展開協力
- YKK APにとってのメリット: 省エネ計算の段階で自社製品が選ばれる = 最上流での顧客獲得

**パナソニック テクノストラクチャー:**
- テクノストラクチャー加盟工務店（約380社）に楽々省エネ計算を推奨ツールとして展開
- 空調・照明・太陽光・蓄電池でパナソニック製品を優先表示
- テクノストラクチャーの構造計算 + 楽々省エネの省エネ計算 = ワンストップ

### 価格設定: ハイブリッドモデル

| プラン | 料金 | 対象 | 含まれる機能 |
|--------|------|------|-------------|
| **都度利用** | 4,980円/回 | 初回利用・小規模事務所 | BEI計算(公式API) + AI製品推薦 + 公式PDF |

### 価格の根拠

**外注との比較（建築士の判断軸）:**
- 外注1件: 50,000〜150,000円
- 楽々都度利用: 4,980円 → **外注の1/10〜1/30**
- 楽々月額Pro: 14,800円/月 → 月に1件でも外注より安い
- 年間10案件の事務所: 外注50万円 vs 楽々月額17.8万円 → **年間32万円の削減**

**SAVE-建築との比較:**
- SAVE-建築: 初年46.2万円 + 年6.6万円
- 楽々月額Pro: 年17.8万円 → **初年で28.4万円安い**

**月額3回以上使えば都度利用より得** → 月額への自然な転換を促す

### サークル会員の位置づけ

| | 外部ユーザー | サークル会員（月5,000円） |
|---|---|---|
| BEI計算 | 4,980円/回 or 月額 | **月3回まで含む** |

## 7. 競合に対する参入障壁（Moat）

| 障壁 | 内容 |
|------|------|
| **メーカーパートナーシップ** | YKK AP会長、パナソニックテクノストラクチャーとの直接関係。他社が容易に真似できない |
| **製品DB** | パートナーから優先提供される性能データ。公開情報+非公開データの組み合わせ |
| **AI推薦の学習データ** | 計算実績が溜まるほど「この条件ならこの製品」の精度が上がる |
| **ネットワーク効果** | 建築士が増える→メーカーが参加したがる→製品が充実→建築士が増える |
| **省エネ義務化の波** | 2025年義務化で需要爆発中。先行者優位が極めて大きい |

## 8. リスクと対策

| リスク | 影響 | 対策 |
|--------|------|------|
| lowenergy.jp API のレート制限 | 大量利用時に制限される可能性 | builelib(MIT)で自前計算エンジンも用意。APIは公式PDF出力時のみ使用 |
| メーカーがアフィリエイトに消極的 | 紹介収益が想定以下 | ツール単体でも月額収益で黒字化可能（粗利95%） |
| SAVE-建築等が同様機能を追加 | 競合激化 | パートナーシップとAI推薦の精度で差別化。先行者優位を活かす |
| 計算結果の正確性への疑義 | 審査機関からの指摘 | 公式API経由の結果のみを提出用とする。自前計算は参考値と明示 |
