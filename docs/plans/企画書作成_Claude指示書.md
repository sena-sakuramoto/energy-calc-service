以下の2つの企画書スライドのテキスト原稿を作成してください。

- 建築業界向けAIサービスの開発者

### サービス概要: 楽々省エネ計算
- 建築士が「製品を選ぶだけ」で省エネ適合（BEI）を確認できるWebツール
- 国交省の公式API（lowenergy.jp）を使用 → 計算結果はそのまま審査提出可能
- AIが建物条件に合った製品を推薦（Claude API）
- 複数メーカーの製品を横断比較できる（YKK AP、LIXIL、三協アルミ、パナソニック、ダイキン等）
- 「見積依頼」ボタンでメーカー・販売店に直接つながる
- 建築士は完全無料で利用可能

### 市場環境
- 2025年4月: 省エネ適合義務化（改正建築物省エネ法）
- 年間44.5万棟が新たに対象（日経クロステック報道）
- 省エネ計算の外注費: 住宅5〜15万円/棟、非住宅30〜100万円以上
- 外注市場はパンク状態（需要 > 供給）
- 既存ツール（SAVE-建築）は初年46万円超と高額
- AI×省エネ計算×製品推薦の一体サービスは市場に存在しない

**スライド1: 表紙**

- 日付: 2026年3月

**スライド2: 省エネ義務化で何が起きているか**
- 2025年4月の義務化で年間44.5万棟が新たに省エネ計算の対象に
- 建築士の悩み: 外注は高い（5〜15万円/件）、自力でやるには専門知識が要る
- 省エネ計算ソフト（SAVE-建築）は初年46万円超
- 国交省の無料ツールはあるが使いにくい
- → 「安く・簡単に・正確に」省エネ計算をしたいニーズが爆発

**スライド3: 建築士はどうやって建材を選んでいるか（現状の課題）**
- 過去の経験 → メーカー営業に連絡 → カタログ取り寄せ → 手動で性能比較
- 省エネ計算は製品選定の「後」に行う → 基準を満たさなければ選定やり直し（手戻り）
- メーカー横断で性能を比較できるツールがない
- → 「製品選定 → 省エネ計算」の順番を逆にすれば、全てが解決する

**スライド4: 楽々省エネ計算とは**
- 「製品を選ぶだけで省エネ適合を確認できる」無料Webツール
- コアフロー図（5ステップ）:
  1. 建物条件を選択（用途・地域・面積）
  2. 製品をカードから選ぶ（AIがおすすめ表示）
  3. 公式APIでBEI計算
  4. 結果表示 + AI改善提案
  5. PDF出力 + 見積依頼
- 公式API使用 → 結果はそのまま審査に提出可能
- 建築士は完全無料

**スライド5: Asutainaとの違い — なぜマルチメーカーが強いか**
- Asutaina for Energy: 優れた簡易計算ツール（YKK AP × one building共同開発）
- ただし: YKK AP製品のみ → 建築士は「比較できない」
- 建築士が本当に欲しいのは「比較した上での最適解」
- 自社専用ツール: 「YKK APの宣伝」に見える
- マルチメーカーで1位表示: 「客観的に優れている」に見える
- → 価格.comに載るほうが自社ECより売れるのと同じ構造
- 楽々省エネ計算はAsutainaと競合しない — 上位レイヤーとして補完する

**スライド6: YKK AP製品が「選ばれる」仕組み**
- AIが建物条件に合った製品を推薦 → YKK AP製品がパートナーとして優先表示
- 建築士は「AIが推薦した＝客観的に最適」と認識
- 省エネ計算の段階で製品が決まる = 最上流での顧客獲得
- 「見積依頼」ボタン → YKK APの営業・代理店にリードが直接届く
- 計算のたびにデータが蓄積:
  - どの地域でAPW430が選ばれているか
  - 競合（LIXIL・三協）との選択比率
  - BEI改善効果の実績データ

**スライド7: YKK APが得られるデータ（イメージ）**
- ダッシュボードのモックアップイメージを記載
  - 月間製品選択回数: APW430: 180回、APW330: 120回、LIXIL TW: 85回…
  - 見積依頼数: 月45件（YKK AP製品）
  - 地域別選択分布: 6地域でAPW430が最多
  - 用途別: 事務所でYKK APシェア65%
- → これは従来のカタログ配布では絶対に取れなかったデータ

**スライド8: パナソニック住宅事業との相乗効果**
- 2026年4月: パナソニック ハウジングソリューションズがYKKグループに合流
- 楽々省エネ計算は窓（YKK AP）+ 空調・照明・太陽光（パナソニック）を横断推薦

- → 加盟店が自力で省エネ適合を確認できれば、設計スピードが上がりコストも下がる

**スライド4: 楽々省エネ計算とは**
- （YKK AP版スライド4と同じコアフロー図）

## トーン・スタイルの指示

- 専門用語は最小限にし、経営者が直感的に理解できる表現を使う
- 数字を多用する（「すごい」ではなく「月200回選ばれています」）
- 1スライドの情報量は抑える（箇条書き3〜5項目が目安）
- 競合批判はしない。自社の優位性を淡々と示す
- 「お願い」のスライドは謙虚に、でも具体的に
