# 楽々省エネ計算 — 無料 vs 有料モデル比較分析

**作成日:** 2026-03-03
**最終更新:** 2026-03-03
**結論:** 有料スタート → スポンサー獲得後に無料切り替え（段階的移行）

---

## 1. 比較対象

| | 有料モデル（旧案） | 無料モデル（採用） |
|---|---|---|
| 建築士への課金 | 4,980円/回 or 月額14,800円〜 | **完全無料** |
| 収益源 | ユーザー課金 + 紹介手数料 | メーカースポンサー + リード課金 + 成約手数料 |
| モデル類型 | SaaS | 両面市場プラットフォーム（価格.com型） |

---

## 4. 無料モデルが勝つ5つの根拠

### ① 競合が既に無料を出している

| 競合 | 料金 | 制約 |
|------|------|------|
| **Asutaina for Energy**（YKK AP） | 無料 | YKK AP製品のみ、非住宅のみ、簡易計算（届出不可） |
| **WEBPRO**（国交省系） | 無料 | UIが悪い、手動入力が多い |
| **外注** | 8〜15万/件 | 1〜2週間かかる |
| **SAVE-建築** | 初年46.2万 | 高額、製品推薦なし |

有料の楽々（4,980円/回）は「Asutainaは無料なのに？」と比較される。
無料の楽々なら「Asutainaより高機能で同じく無料」になる。

### ② trussの成功が証明済み

[truss](https://truss.because-web.com/) = 建材比較・選定プラットフォーム:
- **建築士は無料** → メーカーがスポンサー
- 2020年1億円調達
- 2023年メディアテック（建材関連メディア大手）が買収
- **建築業界で「建築士無料 + メーカー課金」モデルが成立することを実証**

### ③ 両面市場のネットワーク効果

```
有料モデル:
  ユーザー増 → 売上増（線形）

無料モデル:
  建築士増 → メーカー参加増 → 製品充実 → さらに建築士増
         → リード増 → メーカーの支払い意欲向上
         → データ蓄積 → AI推薦精度向上 → さらに建築士増
```

無料モデルは**飛び道具（ネットワーク効果）**を持つ。
有料モデルはどこまでいっても「単価 × ユーザー数」の足し算。

### ④ YKKグループ統合のタイミング（2026年4月）

- YKKがパナソニック住宅ソリューションズ(PHS)を80%取得
- テクノストラクチャー（加盟工務店380社）がYKKグループ傘下に
- 合計売上8,000億円+の巨大グループ

**有料モデルの場合:**
- テクノストラクチャー加盟店に「月額14,800円のツール」を推奨 → 抵抗あり
- 「うちの加盟店に課金するのか？」とグループ内で摩擦

**無料モデルの場合:**
- テクノストラクチャー加盟店に「完全無料のツール」を推奨 → 即採用
- YKKグループ製品が自然にAI推薦される → グループ戦略と完全一致
- 加盟店380社が一気にユーザーになる → メーカー交渉力が一気に上がる

### ⑤ メーカーの方が支払い能力が圧倒的に高い

| | 建築士（個人事務所） | メーカー（YKK AP等） |
|---|---|---|
| 年間売上規模 | 500万〜3,000万 | 4,500億円（YKK AP単体） |
| 月5万円の負担感 | かなりの出費 | 誤差以下 |
| 決裁プロセス | 代表が即判断（コスト意識高い） | マーケ予算から出す（ROI合えば即OK） |
| 楽々から得られる価値 | 省エネ計算の効率化 | **設計段階で自社製品が選ばれる**（最上流のリード獲得） |

メーカーにとって「自社製品が設計時に選ばれる」= 最高のマーケティング。
月50〜100万のスポンサー料は広告費として安い。

## 5. 有料モデルのリスク

| リスク | 影響 |
|--------|------|
| ユーザー獲得が遅い | 有料の壁で初期ユーザーが集まらない → メーカー交渉ができない |
| Asutainaとの比較 | 「YKK APは無料で出してるのに？」と言われる |
| テクノ展開で摩擦 | 加盟店に課金は関係性を損なう可能性 |
| 外注との比較だけが価値 | 「外注より安い」は差別化として弱い |
| 価格競争に巻き込まれる | 後発が安値で参入するリスク |

---

## 6. 無料モデルのリスクと対策

| リスク | 対策 |
|--------|------|
| メーカーが払ってくれない | YKK AP会長・パナソニックテクノとの人脈で初期2社を確保。ツール単体でも原価が低い（200〜350円/計算）ため赤字にはならない |
| ユーザーが来ない | 外注5-15万円 vs 無料 → 「使わない理由がない」状態。テクノストラクチャー展開で一気に380社 |
| 収益化が遅い | Phase 1（MVP）は原価ほぼゼロ。メーカーダッシュボードはPhase 2。スポンサー交渉はMVP稼働後すぐ開始 |
| 「無料=品質低い」の印象 | 国交省公式APIを使用 → 計算結果はそのまま審査提出可能。品質の心配は不要 |

---

## 7. 結論

**無料モデルを採用する3つの決定的理由:**

1. **収益が実は高い** — メーカー課金の方が単価が大きく、月200万 vs 115万で85万円の差
2. **ネットワーク効果** — 両面市場は指数関数的に成長する。SaaS課金は線形成長
3. **YKKグループ統合のタイミング** — 2026年4月。無料なら加盟店380社に即展開。有料では摩擦

**「無料」は弱さではなく、最強の参入障壁。**
後発が楽々に勝つには「無料 + 品質上位」が必要 → 極めて困難。

## 8. 致命的な盲点: キャッシュフロー

上記の分析（Section 1〜7）は「12ヶ月後にどちらが儲かるか」を比較している。
**しかし「12ヶ月後まで生きられるか」を完全に無視していた。**

### 無料モデルの収益化タイムライン

```
Month 1-2:  Phase 1 MVP開発（売上ゼロ）
Month 3:    ローンチ、ユーザー獲得開始（売上ゼロ）
Month 4-5:  Phase 2 メーカーダッシュボード開発（売上ゼロ）
Month 5-6:  YKK AP・パナソニックにスポンサー提案
            → 「ユーザー数は？」→ 稟議開始
Month 7-8:  稟議通過 → 契約 → 初入金
```

**最短でも5〜6ヶ月間、楽々省エネ計算の売上はゼロ。**

trussが初日から無料で走れたのは**1億円の資金調達があったから**。
Senaにはその資金がない。設計受託の売上だけで生活費・税金を払い続ける必要がある。

### 「会長が知り合い」→「月50万の契約」の現実

```
1. 会長に構想を説明する
2. 会長が「面白いね」と言う
3. 会長がマーケティング部門に「検討しろ」と降ろす
4. 部門がROIを検証する
5. 「ユーザー数は？」と聞かれる
6. 「まだゼロです」
7. 「ユーザーが増えたらまた来てください」
```

大企業のスポンサー契約は、トップの「いいね」だけでは通らない。**数字が要る。**

## 9. 最終結論: 有料→無料スイッチ戦略

長期戦略は無料モデルで正解（Section 1〜7の分析はそのまま活きる）。
**変えるのは「いつ無料にするか」だけ。**

### フェーズ設計

```
Phase A: 有料ローンチ（Month 1〜6）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  月額 ¥9,800 で建築士に売る

  なぜ ¥9,800 で売れるか:
  ・外注 8〜15万/件 → 月1件でも元が取れる
  ・SAVE-建築 初年46.2万 → 年間11.8万は1/4
  ・「使わない理由がない」はほぼ変わらない

  月20社 × ¥9,800 = 月19.6万円のキャッシュ
  月50社 × ¥9,800 = 月49万円のキャッシュ
  → 開発を続けながらキャッシュを作る

Phase B: スポンサー獲得（100社到達時）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  YKK AP・パナソニックにスポンサー提案

  「100社の設計事務所が月額課金して使っています。
   御社の製品を優先表示しませんか？」

  100社の課金ユーザー = 「お金を払ってでも使いたい」の証明
  → 無料で集めた100社より説得力が圧倒的に上
  → マーケ部門の稟議が通る

Phase C: 無料スイッチ（スポンサー契約後）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  スポンサー契約が決まったら、建築士側を無料に切り替え

  ・既存有料ユーザーに「無料になりました」と通知
  ・離脱ゼロどころか口コミで一気に拡散
  ・テクノストラクチャー380社展開をここで開始
  ・ネットワーク効果が発動 → Section 3-7の世界へ
```

### なぜこの順番が強いか

| 観点 | 初日から無料 | 有料→無料スイッチ |
|------|-------------|------------------|
| キャッシュフロー | 5-6ヶ月ゼロ | **月1から収益** |
| スポンサー交渉 | 「ユーザーゼロです」 | **「100社が課金中」** |
| ユーザーの質 | 無料だから使ってる | **お金を払う本気の事務所** |
| 無料化の効果 | 最初から無料（当たり前） | **「値下げ」ではなく「無料化」→ ニュース性あり** |
| 資金調達の代替 | VCの金が必要（trussは1億調達） | **自力のキャッシュで代替** |
| リスク | 赤字が続く不安 | **黒字運営しながら拡大** |

### サークル会員の位置づけ（有料期間中）

| | 一般ユーザー | サークル会員（月5,000円） |
|---|---|---|
| 楽々省エネ計算 | 9,800円/月 | **含む（追加料金なし）** |
| 計算回数 | 無制限 | 無制限 |
| AI製品推薦 | 含む | 含む |

サークル会員にとって: ¥9,800のツールが¥5,000に含まれる → **サークルの価値が劇的に上がる**
無料化後: サークル特典は優先サポート + 全ツールバンドルに移行
