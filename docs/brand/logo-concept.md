# 楽々省エネ計算 ロゴコンセプト

このロゴの意味（コンセプト）は、かなり抽象度高めに「省エネ × 建築 × 計算」を1つの記号に圧縮しています。

## 二重の“流れ” = エネルギーの最適化

右に伸びる2本の曲線（葉にも風にも見えるライン）は、
「熱・空気・電力などの“エネルギーが流れる”状態」を抽象化しています。

単に省エネ（我慢）ではなく、流れを整えてロスを減らす = 最適化のニュアンスです。

## 中央のネガティブスペース = 削減されたロス（余白）

曲線の間にできる細い“抜け”は、
ムダが削られていく“余白” = 削減できたエネルギーロスを表現しています。

計算で無駄を見つけて「スッ」と抜く感覚を示しています。

## 左のシャープな多角形 = 建築（構造 / 屋根 / ボリューム）

左側の硬い形は、建築の断面・屋根・マスの記号です。

曲線（環境）に対して、直線的で硬質な形（建築）を組み合わせることで、
建築と環境の両立を示しています。

## 上に向かう重心 = 性能向上（改善・評価向上）

全体のベクトルが右上に伸びるため、
「性能向上」「改善」「最適解に向かう」
= 計算結果が良くなる印象を自然に作れます。

## 配色の解釈

- ベースのチャコール: 信頼・精度・堅実（計算 / 法規 / 実務）
- アクセントのグリーン（またはブルー）:
  - グリーン = 省エネ / 環境
  - ブルー = 技術 / 計測 / データ

どちらでも成立するように、形自体は「環境 × 技術」の中立な記号にしてあります。

## 要約

「建築（硬い形）に、エネルギーの流れ（曲線）を重ね、ロスを抜いて最適化する」

この思想を、文字なしの抽象シンボルに落とし込んだロゴです。
